# EasyOCR Setup (No API Keys Required!)

## Installation:

```bash
cd backend
pip install easyocr googletrans==3.1.0a0
```

## Restart Backend:

```bash
python main.py
```

## Features:

✅ **100% Free** - No API keys needed  
✅ **Works Offline** - Runs on your machine  
✅ **Tamil Support** - Excellent Tamil recognition  
✅ **Auto Translation** - Tamil to English  
✅ **Better than Tesseract** - Deep learning based  

## How It Works:

1. Upload Tamil handwritten/printed image
2. EasyOCR extracts Tamil text (runs in backend)
3. Google Translate converts to English
4. Both texts displayed in frontend

## Note:

First run downloads Tamil language model (~50MB). Subsequent runs are fast!
