# Tamil to English Translator

A clean and modern web application for translating Tamil text to English.

## Setup

1. Get a free API key from [Google AI Studio](https://makersuite.google.com/app/apikey)

2. Create a `.env` file in the frontend directory:
   ```
   REACT_APP_GEMINI_API_KEY=your_actual_api_key
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the application:
   ```
   npm start
   ```

## Usage

- Enter any Tamil text (single letter, word, or paragraph) in the text area
- Click "Translate to English" button
- View the English translation below

## Features

- Clean, modern UI
- Supports single letters, words, and paragraphs
- Real-time translation
- Responsive design
