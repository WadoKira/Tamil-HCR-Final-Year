# Tesseract OCR Setup (Open Source)

## Installation:

```bash
cd frontend
npm install tesseract.js
npm start
```

## Features:

- **100% Free & Open Source** - No API keys needed
- **Works Offline** - Runs entirely in the browser
- **Tamil Support** - Built-in Tamil language recognition
- **Progress Indicator** - Shows OCR progress percentage
- **Toggle Option** - Switch between Tesseract OCR or Local Model

## How It Works:

1. Upload Tamil handwritten/printed image
2. Tesseract extracts Tamil text
3. Automatically provides transliteration
4. No external API calls required

## Note:

Tesseract works best with:
- Clear, high-contrast images
- Printed text (handwriting accuracy may vary)
- Well-lit photos
- Minimal background noise
